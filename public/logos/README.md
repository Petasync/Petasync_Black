# Logos

Platzieren Sie Ihre Logo-Dateien hier:

- `logo.svg` - Haupt-Logo als SVG
- `logo-icon.svg` - Icon-Logo als SVG
- `favicon.ico` - Favicon

Diese Dateien können dann in den Admin-Einstellungen hochgeladen werden oder direkt verwendet werden.

## Verwendung

1. Kopieren Sie Ihre Logo-Dateien in diesen Ordner
2. Commit und push Sie die Änderungen zu GitHub
3. Die Logos werden dann mit dem Code deployed
